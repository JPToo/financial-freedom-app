# Upload Guide

1. Unzip the file.
2. Upload these 4 files to GitHub:
   - app.py
   - requirements.txt
   - README.md
   - STEP_BY_STEP_GUIDE.md
3. Commit changes.
4. Streamlit main file path stays:
   app.py
5. Wait for Streamlit to redeploy, or reboot the app.

Phone:
Use the same Streamlit URL. Refresh the phone browser after redeployment.
